Sora is an open-source portfolio template designed for developers, researchers, and creators who want to showcase their life's work in a clean, professional manner.

## Features

- **Single-page board** with filter tabs for different content types
- **Modal-based UX** — click to view details without page navigation
- **Multiple content types** — OSS contributions, projects, writing, talks, certifications
- **Dark/light mode** with system preference detection
- **Fuzzy search** powered by Pagefind
- **Fully configurable** — single config file for all personal data
- **SEO optimized** with dynamic OG image generation

## Tech Stack

Astro v5, TypeScript, Tailwind CSS, native HTML `<dialog>` element.
