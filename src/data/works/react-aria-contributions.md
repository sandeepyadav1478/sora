Contributed to React Aria, Adobe's library of unstyled, accessible UI primitives used by thousands of applications.

## Contributions

- Improved keyboard navigation for `ComboBox` and `Select` components
- Fixed ARIA attribute handling in `DatePicker` for screen readers
- Added comprehensive test coverage for focus management utilities
- Helped triage accessibility-related issues from the community

## Impact

React Aria is used across Adobe's product suite and by the broader React community. These contributions directly improved the experience for users who rely on assistive technologies.

## What I Learned

Working on accessibility at this scale taught me that a11y isn't a feature — it's a fundamental requirement. The patterns I learned here now inform every component I build.
